# Legacy warping results (archived)

This folder stores evaluation artifacts produced by the older warping pipeline. They are kept for reference only and are not used by the current lung-only warping flow.

- `metrics/audit_results.json`: QA audit summary for the legacy warped dataset.
- `metrics/experiment_results.json`: Aggregate metrics from warping experiments.
- `validation/cross_evaluation_valid_3classes/`: Cross-evaluation results for the 3-class warped validation set.
- `validation/pfs_warped_valid/`: PFS validation results for legacy warped samples.
- `validation/external_validation/warped_on_warped_results.json`: External validation results for warped-on-warped evaluation.
- `detailed_analysis/`: Image-level TP/TN/FP/FN examples from the legacy evaluation.
